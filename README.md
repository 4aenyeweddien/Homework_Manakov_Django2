# Домашняя работа по блоку 23

## Описание:

Домашняя работа по блоку 23. Работа с ORM в Django. Произведено подключение СУБД PostgreSQL. Внесены изменения в 
настройки подключения в файле "settings.py". В приложении каталога созданы модели Product и Category и описаны для них 
базовые настройки. Перенесены отображения моделей в базу данных с помощью инструмента миграций. Создан суперпользователь,
Модели Product и Category зарегистрированы в админке и настроены отображения моделей согласно ТЗ. Заполнены через 
инструменты shell список категорий и реализованы фильтры. Сформированы фикстуры в формате json и реализована кастомная
команда, которая удаляет данные из БД и добавляет данные из фикстур.


## Установка:

1. Клонируйте репозиторий:
```
git clone https://github.com/username/myblog.git
```
2. Установите зависимости:
```
pip install -r requirements.txt
```
