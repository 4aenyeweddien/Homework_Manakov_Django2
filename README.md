Домашняя работа по блоку 22. Знакомство с Django. Инициализирован проект Django и создано приложение catalog. Проведена
регистрация приложения в настройках проекта. Настроена маршрутизация для нового приложения.
Созданы html шаблоны home и contacts. Подключены css-стили и js-скрипт bootstrap. Так же реализована форма обратной 
связи на странице контактов и настроена обработка данных в контроллере, для отображения сообщений об успешной отправке 
сообщения.
